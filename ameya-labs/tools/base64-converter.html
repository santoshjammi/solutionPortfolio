What is Base64?

How Base64 Works

Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It encodes every 3 bytes of binary data into 4 ASCII characters.

  • • Uses 64 different ASCII characters (A-Z, a-z, 0-9, +, /)
  • • Increases data size by approximately 33%
  • • Commonly used for email attachments and data URIs

Common Use Cases

  • • Embedding images in HTML/CSS
  • • API authentication (Basic Auth)
  • • Storing binary data in text files
  • • Email attachments
